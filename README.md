# coronaApp
Small web application to Scrap data to Front End Page

```sh
git clone git@github.com: enghamzasalem/coronaApp.git # or clone your own fork
cd coronaApp
npm install
node index.js
```

Your app should now be running on [localhost:3055](http://localhost:3055/).

## Deploying to Heroku

```
heroku create
git push heroku master
heroku open
```
